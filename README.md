# 📊 Estudos de Pandas com Python

Este repositório faz parte da minha jornada de aprendizado em **análise de dados com Python**, com foco no uso da biblioteca **Pandas** para manipulação, limpeza e análise de dados.

O objetivo é desenvolver habilidades práticas utilizadas por **Analistas de Dados, Cientistas de Dados e Engenheiros de Dados**.

---

# 🐍 Tecnologias utilizadas

- Python  
- Pandas  
- NumPy  
- Jupyter Notebook / VS Code  

---

# 📚 Conteúdos estudados

Este repositório contém exercícios e pequenos projetos abordando:

### 📥 Manipulação de dados
- Leitura de arquivos CSV
- Leitura de arquivos Excel
- Importação de datasets

### 🧹 Limpeza de dados
- Tratamento de valores nulos
- Remoção de duplicados
- Conversão de tipos de dados

### 📊 Análise de dados
- Estatísticas descritivas
- Média, mediana e variância
- Agrupamentos (`groupby`)
- Filtros e seleções

### 🔎 Exploração de dados
- Ordenação de dados
- Criação de colunas
- Transformações de datasets
